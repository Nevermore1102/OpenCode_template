## What I do
- 在GitHub仓库中搜索文档和问题
- 读取GitHub仓库中的特定文件
- 探索GitHub仓库的目录结构
- 深入探索开源项目

## When to use me
Use this when you need to:
- 研究开源项目
- 查看GitHub仓库的文件内容
- 了解项目结构
- 搜索项目文档和issue

## Available workflows
1. **search-repo-docs**: 在GitHub仓库中搜索文档和问题
2. **read-github-file**: 读取GitHub仓库中的特定文件
3. **explore-repo-structure**: 探索GitHub仓库的目录结构
4. **deep-dive-repo**: 深入探索仓库：获取结构 -> 搜索文档 -> 读取关键文件

## Examples
- 搜索Vue.js仓库关于组件生命周期的问题
- 阅读Vite项目的package.json文件
- 探索Next.js仓库的目录结构
- 研究Tailwind CSS的构建配置
