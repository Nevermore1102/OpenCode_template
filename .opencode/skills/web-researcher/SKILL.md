## What I do
- 搜索网络信息
- 获取并解析网页内容
- 研究特定技术主题
- 对比多个来源的信息

## When to use me
Use this when you need to:
- 搜索最新的技术资讯
- 获取网页的完整内容
- 深入研究某个技术主题
- 对比不同来源的信息

## Available workflows
1. **search-web**: 搜索网络信息
2. **fetch-webpage**: 获取并解析网页内容
3. **research-topic**: 研究特定主题：搜索 -> 获取网页 -> 分析内容
4. **compare-sources**: 对比多个来源的信息

## Examples
- 搜索2025年最新的前端框架
- 获取这个技术文档的完整内容
- 研究React Server Components
- 对比不同CI/CD工具的优缺点
